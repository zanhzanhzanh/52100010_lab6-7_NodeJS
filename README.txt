Lab sử dụng MySQL workbench và Sequelize
1. Cài đặt sequelize trên local (sử dụng npm/yarn)
2. Tạo một connection mới trên MySQL workbench (Sửa lại file config.json nếu có password)
3. Dẫn terminal vào đường dẫn NodeJS-ThucHanh\Lab06\src
4. Dùng lệnh sequelize db:create tạo DB
5. Dùng lệnh sequelize db:migrate
6. Dẫn terminal vào đường dẫn NodeJS-ThucHanh\Lab06\
7. Chạy bằng lệnh yarn start
